# Knowledge base: source code / error code dumps

Drop vendor source snippets, error code tables, MIBs, log samples here.

Examples:

- `cisco_ap_firmware_3.2.1.c`
- `onvif_error_codes.txt`
- `biostar_event_codes.csv`

Files are weighted slightly lower than resolution docs in `errors/`,
but they help when the LLM needs to interpret an unfamiliar
`status_code`.
